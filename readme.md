## Run Server

    go run cmd/wonderwall/main

## Run App

    go run wonder/main

## Web build

Install js dependencies

```bash
npm install
```

build js assets

```bash
webpack --env.NODE_ENV=local --progress

webpack --env.NODE_ENV=dev --env.production=false --progress
```
